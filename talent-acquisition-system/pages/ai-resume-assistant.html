AI Resume Assistant

Resume Analysis Chatbot

Welcome to the AI Resume Assistant powered by Gemini. I can help you find the best candidates for your open positions by analyzing all available resumes. Please provide the position title and description of the candidate you're looking for.

How to Use
  1. Make sure you have uploaded resumes using the Upload Resume page
  2. Describe the position and candidate requirements (e.g., "I'm looking for a Frontend Developer with React experience")
  3. The AI will analyze all available resumes in the system
  4. Review the AI's recommendations based on resume match scores
  5. Ask follow-up questions about specific candidates
  6. Ask about eliminated candidates with "Show me the eliminated candidates"
Important: Always start by describing a position first, then ask about candidates.
Example Prompts
  • Find candidates for a Senior Frontend Developer with 5+ years of React experience
  • Why was candidate John Smith eliminated?
  • Compare the top 3 candidates for this position
  • What skills are missing from the recommended candidates?
